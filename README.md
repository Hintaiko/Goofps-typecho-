# Goofps

> Typecho 实时 FPS 帧率显示插件，兼容 PureSuck 主题深浅色模式。

## 版本

1.0.0

## 简介

在网站前台固定位置实时显示 FPS（每秒帧数）指示器。支持自定义前景颜色、背景颜色、桌面/手机端字号、顶部/左侧位置、圆角、阴影、状态点和刷新频率。

通过 `html[data-theme="dark"]` 属性自动切换深色模式配色，同时兼容系统级 `prefers-color-scheme` 媒体查询。

---

## 安装

1. 将 `Goofps` 目录放入 Typecho 的 `usr/plugins/` 目录下：
   ```
   usr/plugins/Goofps/
   └── Plugin.php
   ```

2. 进入 Typecho 后台 **控制台 → 插件**。

3. 找到 **Goofps**，点击 **启用**。

4. 点击插件右侧的 **设置** 进行自定义配置。

## 设置项

| 设置项 | 类型 | 默认值 | 说明 |
|-------|------|--------|------|
| FPS 颜色 | 下拉选择 | 素粉 | 文字、状态点和边框颜色，支持 6 种预设 |
| 背景颜色 | 下拉选择 | 跟随网站背景 | 浅色/深色模式自动适配 |
| 电脑端字号 | 数字输入 | 13 | 单位 px，范围 9-30 |
| 手机端字号 | 数字输入 | 11 | 单位 px，范围 9-24 |
| 顶部距离 | 数字输入 | 16 | 单位 px，范围 0-500 |
| 左侧距离 | 数字输入 | 16 | 单位 px，范围 0-500 |
| 边框圆角 | 数字输入 | 10 | 单位 px，范围 0-50 |
| 阴影 | 开关 | 显示 | 浅色/深色模式自动切换阴影深度 |
| 状态点 | 开关 | 显示 | FPS 值左侧的彩色圆点指示器 |
| 刷新频率 | 下拉选择 | 0.5 秒 | 仅影响数字刷新速度，不影响统计准确性 |

## 深色模式兼容

PureSuck 主题使用 `html[data-theme="dark"]` 切换深色模式。本插件通过以下 CSS 规则自动适配：

```css
html[data-theme="dark"] #goofps {
    --goofps-accent: /* 深色前景色 */;
    --goofps-background: /* 深色背景色 */;
}

@media (prefers-color-scheme: dark) {
    html:not([data-theme]) #goofps {
        /* 系统级深色模式回退 */
    }
}
```

## 技术细节

- **FPS 统计**：基于 `requestAnimationFrame` 精确计算，不受刷新频率设置影响。
- **后台省电**：页面不可见时自动暂停统计，切换回标签页后自动恢复。
- **性能**：`contain: layout style paint` 限制重绘范围，`pointer-events: none` 不干扰页面交互。
- **无障碍**：使用 `role="status"` 和 `aria-label` 标注，支持屏幕阅读器。
- **打印**：打印时自动隐藏 FPS 指示器。
